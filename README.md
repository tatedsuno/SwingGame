# スイングゲーム（箱庭）

Three.js 製のスイングアクションゲーム。ビルや地面にロープを発射して移動します。

## プレイ

**https://tatedsuno.github.io/SwingGame/**

## 遊び方

1. 上のリンクを開く（ローカルでは `swing-game.html` をブラウザで開いても可）
2. 左クリック：ワイヤー発射（押し続けでスイング）／離す：手放す
3. スイング中：W/Sで巻き取り調整、A/Dで左右操舵
4. 地上・空中：WASDで移動、Space：ジャンプ

## ファイル

- `swing-game.html` — ゲーム本体（単一 HTML ファイル）
